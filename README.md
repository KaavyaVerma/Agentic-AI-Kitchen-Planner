# Agentic-AI-Kitchen-Planner
🍽️ AI-Powered Weekly Meal Planner

An AI-powered Weekly Meal Planner built for Google Colab that helps you create personalized meal plans using the ingredients you already have at home.

Simply upload your kitchen inventory CSV files, specify your dietary preferences, and let the AI generate a complete 7-day meal plan, including nutrition insights, ingredient usage tracking, and a shopping list for missing items.

Powered by Meta-Llama models through Hugging Face Inference API.

*Features*
- Upload kitchen inventory data via CSV files
- AI-generated personalized 7-day meal plans
- Supports dietary preferences and restrictions
Vegetarian
Vegan
High-Protein
Low-Carb
Gluten-Free

*Custom preferences*
- Nutrition analysis for each meal and daily totals
- Ingredient quantity tracking across the week
- Automatically generated shopping list for missing ingredients
- Runs directly in Google Colab
- Powered by Meta-Llama through Hugging Face
